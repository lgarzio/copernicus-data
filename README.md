# copernicus-data
Quick method of downloading data from the E.U. Copernicus Marine Service Information (CMEMS): [Global Ocean Physics Reanalysis (GLORYS12V1)](https://data.marine.copernicus.eu/product/GLOBAL_MULTIYEAR_PHY_001_030/description) dataset using the Python Copernicus Marine Toolbox API. In order to use the API you must have your Copernicus Marine credentials either stored in a configuration file or by declaring your username and password arguments in the function (see [documentation](https://help.marine.copernicus.eu/en/articles/8286883-copernicus-marine-toolbox-api-get-original-files) for details)
